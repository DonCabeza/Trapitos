---

## Ejecutar

### Paso 1: Clonar el Repositorio
El proyecto debe vivir dentro de la carpeta pública de tu servidor local. 
1. Abre tu terminal.
2. Navega hasta la carpeta `htdocs` de tu instalación de XAMPP.
3. Ejecuta el siguiente comando para descargar el código:

```bash
cd C:/xampp/htdocs/
git clone [https://github.com/DonCabeza/Trapitos.git](https://github.com/DonCabeza/Trapitos.git) MisTrapos/Trapitos
```

### Paso 2: Ejecuten todo
1. Abrir xampp y eejcutar apache
2. Tener lanzado el servidor psql
3. Tengan las tablas, si no ejecuten los archivos alter_ventas y trapitos_db
4. Abran la ruta donde tengan el proyecto en el navegador http://localhost/index.php si ponen dentro de una carpeta pues a partir de localhost/ navegan.
